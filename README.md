# NEOMAIND
Сервис для приема и обработки заявок от клиентов на ремонт и дизайн квартир
, а также планирование и заказ изделий для них. Сервис 
состоит из 2х компонентов - API и суперадминка.


## Start app
```bash
docker-compose up
```
